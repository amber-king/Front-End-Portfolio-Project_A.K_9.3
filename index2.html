About the Webpage

This webpage is to help find full address information from all over the world via [zippopptam.us API](https://www.zippopotam.us/?format=j1).

User will recieve the following information as a result:

- city(place name)

- state

- country

- postal code (zip code)

- state abbreviation

- country abbreviation

How to Search Location?

(1) The user must put in one or more of the following information into the top search engine on the page next to "What's your location?" to receive valid results:
- city
- state
- country
- zip code

Format Guide:

The two formats to type in the search engine for information to return a full address are:

country/zip code

OR

country/state/city

(2) Click the "Get Address!" and your resutls will appear under the search engine. This information will be your City, State, Country, State Abbreviation and Country Abbreviation.

**NOTE**
Some locations state, state abbreviation and postal code/zipcode details are undefined if the city input found in other parts of the world. Ex:When you search by country/state/city the state and postal code/zip code imformation is undefined due to the city being in other states besides the desired search city.
